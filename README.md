# 八字日记应用

一个结合传统命理学的日记应用，帮助您记录每日运势并分析五行喜忌。

## 功能特点

- 📅 **自动计算八字**：自动显示每日的年月日三柱，五行颜色标注
- 📝 **运势记录**：记录每日吉/平/凶的运势体感和心情
- 📊 **数据分析**：按月统计分析，帮助判断喜用神
- 📱 **移动适配**：响应式设计，完美支持手机端使用
- ☁️ **云端同步**：基于 Supabase，数据安全可靠

## 技术栈

- **前端**：HTML + CSS + JavaScript
- **八字计算**：lunar-javascript
- **数据库**：Supabase (PostgreSQL)
- **部署**：可直接在浏览器打开使用

## 快速开始

1. 直接在浏览器中打开 `index.html` 即可使用
2. 记录运势：选择今日运势、填写心情记录
3. 查看统计：切换到"统计"页面，选择月份进行分析

## 使用说明

### 记录运势

1. 打开应用，查看今日八字（自动显示）
2. 选择今日运势：吉/平/凶
3. 填写心情记录（可选，最多800字）
4. 点击"保存记录"

### 统计分析

1. 切换到"统计"页面
2. 选择要分析的月份
3. 点击"分析"查看结果
4. 查看吉凶日的天干地支五行分布
5. 参考智能提示了解可能的喜用神

## 数据库

项目使用 Supabase 作为后端存储：

- **项目名称**：journal-app
- **表名**：journal_entries
- **数据安全**：启用了 RLS 策略
